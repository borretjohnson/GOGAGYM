# 🩸 GOGAGYM MINI APP

**Telegram Mini App для фитнес-канала** — твой путь воина!

---

## 🔥 ФУНКЦИОНАЛ

| Раздел | Что внутри |
|--------|------------|
| 🔥 **Тренировка** | Таймер, 5 раундов, упражнения, прогресс бар |
| 📊 **Прогресс** | Вес, замеры тела, история, календарь тренировок |
| 🎯 **Цели** | Выбор цели (сжигание/масса/выносливость/сила), чек-лист на день |
| 💧 **Вода** | Трекер потребления воды с визуализацией |
| 🧮 **Калькулятор** | Расчёт калорий (BMR/TDEE), дефицит/профицит |
| 🏋️ **Упражнения** | База из 8 упражнений с описанием |
| 📝 **Заметки** | Личные записи, мысли, советы |
| 📊 **История** | Все тренировки и замеры |
| 🏆 **Достижения** | 6 достижений для разблокировки |
